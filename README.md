# AI 商业化认知库

系统性记录和整理 AI 商业化相关的认知、洞察与实践经验。

## 目录结构

```
├── business-models/      # 商业模式：ToB、ToC、API、平台等
├── market-analysis/      # 市场分析：行业趋势、竞争格局、市场规模
├── product-strategy/     # 产品策略：定位、定价、PMF、GTM
├── case-studies/         # 案例研究：成功/失败案例复盘
├── tech-trends/          # 技术趋势：模型演进、基础设施、开源生态
├── investment/           # 投融资：估值逻辑、融资趋势、退出路径
└── notes/                # 随笔与碎片化思考
```

## 使用方式

- 每篇文档以 Markdown 格式撰写
- 文件名格式：`YYYY-MM-DD-标题.md`
- 欢迎持续补充和修订
